Chillow: A Zillow Clone

Built using: Ruby on Rails, React(w/ Redux), Postgres, Active Record, HTML, CSS

When users arrive to Chillow, they can sign-in, peruse some suggested listings in the carousel at the bottom, or begin their search for a quaint winter cabin with a simple keyword search using the search bar in the middle. Once a user is already signed in, the splash page will offer them the ability to view their saved listings and saved searches (as pictured above).

Users can create an account using the sign-up modal or toggle to sign-in if they already have an account. Passwords, which are encrypted using bcrypt, must satisfy the constraints in red font above (constraints dynamically turn green as constraints are met). Visitors of Chillow are welcome to sign-in as a 'Demo user' by clicking said button at the bottom of the sign-in modal.

I am proud of how similarly I was able to clone the listing's show page. I learned a tremendous amount of CSS in this component. The images on the left scroll independently of the listing details, the property description is expandable/collapsable, and right column can be toggled between an 'Overview' and a 'Facts and features' sections.

The map was built with Googel Maps API service and dynamically displays the pinned location of each property listing. Pins display prices and can be clicked to view the listing's show page. Above, users can apply various search parameters to modify their search (and save search if desired), which is relflexted on both the map as well as the listing index on the right.

Similar to Zillow, users can not only search listings using filters, but also save these search filter parameters to be revisited in the future. On the user's saved searches page, they can edit or remove certain search parameters that they have archived. In the future, I intend to add more rubust filtration parameters such as geographical boundaries.
